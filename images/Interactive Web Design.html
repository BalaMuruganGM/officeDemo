The property align-items: space-around; does not work because align-items does not accept space-around as a valid value.

💡 Why?

  • align-items is used to align items along the cross-axis (vertically in a row and horizontally in a column).

  • It only accepts values like:

    • stretch (default)
    • flex-start
    • flex-end
    • center
    • baseline

  • space-around is only valid for justify-content, which controls horizontal spacing in a flex container.

✅ Correct Usage

❌ Wrong (won't work)

css
.container {
    display: flex;
    align-items: space-around; /* ❌ Invalid */
}

✅ Correct (use justify-content instead)

css
.container {
    display: flex;
    justify-content: space-around; /* ✅ Works for horizontal spacing */
    align-items: center; /* ✅ Centers items vertically */
}
